<!-- GETTING STARTED -->
## 入门

克隆仓库代码。:smile:

### 依赖安装

1. 安装 [Node.js](https://nodejs.org/)
2. 安装 [ANTLR 4](https://www.antlr.org/)

### 安装

1. 克隆项目代码
   ```sh
   git clone https://github.com/w158rk/vue-echart-map.git
   ```
2. 安装npm包
   ```sh
   npm install
   ```
3. 构建项目
   ```sh
   npm run build
   ```

## 使用

构建项目之后在`dist/`目录下运行http服务器，例如，可以使用`npx serve .`运行服务。
